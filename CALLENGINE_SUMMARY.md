# CallEngine - Собственная библиотека для звонков

## 📋 Обзор

Создана полнофункциональная библиотека для управления аудио/видео звонками в мессенджере на базе WebRTC.

## 🚀 Быстрый старт

### Установка

Файлы уже созданы в проекте. Просто импортируйте:

```jsx
import { useCallEngine } from './hooks/useCallEngine';
```

### Использование

```jsx
function MyComponent({ currentUser }) {
  const {
    callState,
    remoteUser,
    startCall,
    answerCall,
    endCall,
  } = useCallEngine(currentUser);

  return (
    <div>
      {callState === 'idle' && (
        <button onClick={() => startCall(targetUser, 'video')}>
          Позвонить
        </button>
      )}
      
      {callState === 'incoming' && (
        <div>
          <p>Звонок от {remoteUser?.username}</p>
          <button onClick={answerCall}>Ответить</button>
        </div>
      )}
      
      {callState === 'active' && (
        <button onClick={endCall}>Завершить</button>
      )}
    </div>
  );
}
```
